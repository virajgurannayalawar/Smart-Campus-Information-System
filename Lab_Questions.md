# Python Programming Laboratory - Smart Campus Information System
**Application Context:** Dayananda Sagar College of Engineering plans to develop a simple Python-based Smart Campus Information System to manage student information, academic records, and performance analysis.

---

## Q1. Student Registration and Grade Evaluation
Design a Python program that accepts student details such as student name and exam score from the user. The program should evaluate the score using conditional statements and determine the grade of the student. Based on the obtained marks, the program should also display an appropriate performance remark such as Excellent, Very Good, Good, Average, or Needs Improvement.

**The program has three main parts:**
1. **Input collection:** Ask the user to enter the student’s name and exam score.
2. **Grade evaluation:** Use if-elif-else conditions to check the score range and assign a grade (A, B, C, D, or F) and remark.
3. **Output display:** Print the student’s name, score, grade, and remark.

## Q2. Course Enrollment Management System
Design a Python program to manage course enrollment for students. The program should allow the administrator to enter course names and corresponding credit values using loops. 

**The program should:**
* Use loops to allow multiple course entries.

* Use the reak statement to limit the maximum number of courses that can be added for a student.
